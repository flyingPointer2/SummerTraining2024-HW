What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS Syntax

CSS is composed of selectors and declarations. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons.

Selectors

At its core, CSS syntax consists of rulesets. A ruleset is made up of a selector and a declaration block.

/* Selector */
element {
    /* Declaration block */
    property: value;
}

Selectors target the HTML elements you want to style. They can be element type, class, ID, or attribute based.

/* Element type selector */
p {
    color: blue;
}

/* Class selector */
.important-text {
    font-weight: bold;
}

/* ID selector */
#header {
    background-color: #eee;
}

Properties and Values

Inside the declaration block, CSS properties define the style characteristics, and values are what those properties are set to.

body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

Inheritance

CSS properties can be inherited from parent elements to their children, reducing redundancy in your stylesheets.

/* Parent element */
div {
    font-size: 16px;
}

/* Child element inherits font size */
div p {
    color: grey;
    /* Inherited font size */
}

The Box Model

Understanding the box model is fundamental to CSS. Every element is a box with content, padding, border, and margin areas.

box {
    width: 300px;
    padding: 20px;
    border: 5px solid #000;
    margin: 10px;
}

Pseudo-classes and Pseudo-elements

Pseudo-classes and pseudo-elements add specific styles for different states or parts of an element.

/* Pseudo-class for links */
a:hover {
    color: red;
}

/* Pseudo-element for first line of a paragraph */
p::first-line {
    font-weight: bold;
}

Media Queries

Media queries are essential for responsive design, allowing styles to adapt based on the device's characteristics.

@media (max-width: 600px) {
body {
    background-color: lightblue;
}
}

CSS Comments

Comments in CSS are used to explain code or temporarily remove code without deleting it.

/* This is a single-line comment */
/* 
This is a multi-line comment
*/

body {
/* Set default font and background */
    font-family: sans-serif;
    background-color: #fff;
}
