Introduction to C++

C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language. It has imperative, object-oriented, and generic programming features, while also providing facilities for low-level memory manipulation.

Basic Syntax

C++ syntax is similar to C, with the addition of object-oriented features such as classes and objects. It also supports multiple paradigms, including procedural, object-oriented, and functional programming.

Setup and Main Function

Every C++ program starts with the inclusion of libraries, using the preprocessor directive #include. The entry point of every C++ program is the main() function, which must return an integer.


#include <iostream>

int main() {
    // Your code here
    std::cout << "Hello world!" << std::endl;
    return 0;
}

Variable Declaration

Variables in C++ must be declared before they are used, specifying their type. The scope of the variable is limited to the block in which it is defined.


int age = 25;
double height = 5.5;

Data Types and Operators

C++ supports a variety of data types including integers, floating points, characters, and more. Operators are used to perform operations and include arithmetic, relational, logical, and assignment operators.


int x = 10;
int y = 20;
int sum = x + y;  // Arithmetic operator

Control Structures

Control structures dictate the flow of a program. C++ provides if-else statements, switch-case, loops (for, while, do-while), and more.


if (x > y) {
    std::cout << "x is greater than y";
} else {
    std::cout << "x is less than or equal to y";
}

for (int i = 0; i < 10; ++i) {
    std::cout << i << " ";
}

Functions

Functions in C++ are blocks of code that perform a specific task. They can take inputs (parameters) and return outputs (return type).


int add(int a, int b) {
    return a + b;
}

int result = add(5, 10);

Arrays and Pointers

C++ allows the use of arrays to store multiple values of the same type and pointers to store memory addresses of variables.


int numbers[] = {1, 2, 3};
int *ptr = numbers;  // Pointer to the first element of the array

Input and Output

C++ provides the iostream library for input and output operations. The std::cout object is used for outputting data, while std::cin is used for input.


std::cout << "Enter your name: ";
std::string name;
std::cin >> name;
std::cout << "Hello, " << name << "!";
